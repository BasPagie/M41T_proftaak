Waarom modules in JS?

In javascript modules zijn kleine delen herbruikbare code. Met deze kleine stukken code kun je een patroon maken in je applicatie om alle code netjes te houden.

Modules worden vooral gebruikt voor deze functie, het overzichtelijk maken van je code en om de communicatie te verbeteren tussen verschillende programmeurs.

Waarom frameworks in JS?

Een framework is een geraamte waarin een bepaalde structuur is opgezet waarop je verder kunt bouwen. Je kunt het zien als een "code-template". Voorbeelden van een template zijn: Backbone.js, Angular.js en Bootstrap.js.

We hebben frameworks nodig om gemakkelijk met een basis applicaties te maken voor een specifiek probleem. Het is ook overzichtelijker om via een framework met een team te coderen. Het bespaart veel tijd en je hoeft minder te coderen.

Waarom libraries in JS?

Een library is een toolkit van code die je kunt gebruiken om van niks iets te maken. Een library bevat functies, methodes en controls die het coderen van applicaties makkelijker maakt. Voorbeelden van library's zijn: Jquery, MooTools en Three.

Libraries kunnen gebruikt worden voor het streamlinen van je code, het besparen van tijd en het coderen & communiceren via een library gemakkelijker is. In een groot team zou het een goede keuze zijn om een library te gebruiken.

Essentie van Frameworks

De belangrijkste rede waarom je frameworks gebruikt kun je met een woord aanduiden: efficiency.

Frameworks geven zo veel mogelijk shortcuts om met zo min mogelijk code en tijd, een volledige applicatie te kunnen maken.

D3.js Library

D3.js is een javascript library voor het maken van visualisaties gebaseerd op (live) data.

D3.js maakt gebruik van DOM manipulatie. Je kunt hiermee bijvoorbeeld een html table genereren van een array van nummers.Je kunt deze data ook gebruiken om een interactieve SVG bar chart met animaties en interactie te maken.

Demo D3.js

Sproutcore Framework

Sproutcore is een clientside framework die vooral zorgt voor een snelle en sterke website waarop de user experience erg goed is.

Textures.js Library

Textures.js is een Javascript library voor het maken van SVG bestanden en het visualiseren van data. De patterns kunnen gecombineerd worden waardoor je eindeloze mogelijkheden hebt. Het is afgeleid van D3.js

Ionic Framework

Ionic is een framework voor het ontwikkelen van hybride mobiele applicaties Het is bovenop AngularJS en Apache Cordova gebouwd.

Voordelen

Levert alle functionaliteiten die native SDK's hebben Zowel voor Android als iOS, builden via Cordova Toegangkelijk voor webdevelopers HTML5/CSS Ionic levert fijn grid systeem, push notifications

Nadelen

Goede kennis van AngularJS nodig (kan ook een voordeel zijn natuurlijk) Niet volledig native, dus niet optimaal voor het maken van games
